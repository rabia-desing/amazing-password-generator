# **Introduction:**

Amazing Password Generator application powered by JavaScript with clean code. You can choose what to include in your password by just clicking OK, cancel from the prompts.

-   UPPERCASE
-   LOWERCASE
-   NUMERIC
-   SPECIAL CHARACTERS
